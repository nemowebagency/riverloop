# Riverloop

Sito web aziendale di Riverloop - Sviluppo Software, Formazione Professionale e Consulenza Digitale.

## Tecnologie Utilizzate

- React 18
- TypeScript
- Tailwind CSS
- React Router DOM
- Vite

## Servizi

- Sviluppo Software (Web, Mobile, Videogiochi, AI, Blockchain)
- Formazione Professionale (Corsi Tech, Certificazioni, Formazione Finanziata)
- Consulenza Digitale

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
